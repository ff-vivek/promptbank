Simple Commands

Everything you need, nothing you don't

promptbank add

Add a new prompt interactively or with flags

promptbank list

List all prompts, filter by category

promptbank get <id>

View a prompt, optionally copy to clipboard

promptbank apply <id>

Render a prompt with variable substitution

promptbank search <query>

Search prompts by name, tags, or content

promptbank community browse

Browse available community prompts

promptbank community install <name>

Install a prompt from the community

promptbank claude install <id>

Install a prompt as a Claude skill

promptbank claude list

List prompts installed in Claude

promptbank update

Update promptbank to the latest version

promptbank export <file>

Export prompts to a JSON file

promptbank import <file>

Import prompts from a JSON file
